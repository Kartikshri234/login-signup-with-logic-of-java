Welcome to the Exam Portal

A simple, responsive demo app that demonstrates signup/login (using localStorage) and a small graded exam. Use the navigation to sign up, log in, and take the exam.

Create Account Sign In

Quick Info

  • No backend required — demo uses localStorage
  • Client-side validation + accessible markup
  • Score: +5 for correct, -2 for wrong (no negative final score)

How to use

  1. Signup with a user id and password.
  2. Login — you'll be redirected to the exam page.
  3. Answer questions and submit to see your score.
Built for demo • No personal data is sent anywhere.
